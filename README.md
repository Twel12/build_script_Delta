# PixelOS build_scripts for davinci

## Usage:
```bash
# create a folder to hold source code
mkdir PixelOS
cd PixelOS

# clone build_scripts into aosp
git clone https://github.com/Twel12/build_script_Delta.git build_scripts

# run build scripts
bash build_scripts/server.sh
```
